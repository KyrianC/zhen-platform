#+TITLE: zhen-platform

* Setup
** Development
Create .env file, it should contain:
+ =BASE_URL=: base url used by axios on frontend (eg: ~http://localchost:8000/api~)
+ =SECRET_KEY=: django secret key
+ =DEBUG=: django debug settings, ~1~ for True, ~0~ for False
+ =DJANGO_SETTINGS_MODULE=: which django settings module to use (eg: ~config.settings.dev~)
+ =POSTGRES_ENGINE=
+ =POSTGRES_USER=
+ =POSTGRES_PASSWORD=
+ =POSTGRES_DB=
+ =POSTGRES_HOST=
+ =POSTGRES_PORT=

Build containers and start them:
#+begin_src shell
docker-compose up --build
#+end_src
If django app start before postgres is initialized stop and restart

Apply django migrations:
#+begin_src shell
docker-compose exec api python manage.py migrate
#+end_src

Create django super user:
#+begin_src shell
docker-compose exec api python manage.py createsuperuser
#+end_src
